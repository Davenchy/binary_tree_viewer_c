# Binary Tree Viewer

- Simple binary tree viewer written in c

## Example

- Check `main.c` for an example

- Output

```bash
$ ./binary_tree
                 .-----------------(010)-----------------.                 
       .-------(005)-------.                   .-------(015)-------.       
     (001)            .--(003)--.            (006)            .--(012)--.  
                    (009)     (011)                         (013)     (014)
```

## Build

```bash
$ make
```
